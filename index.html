📚 Glossary ความมั่นคงสารสนเทศ

🔐 Confidentiality (ความลับ)

คืออะไร : จำกัดสิทธิ์การเข้าถึงข้อมูลเฉพาะผู้ได้รับอนุญาต

ถ้าเจอปัญหาแบบนี้ :
📌 แชร์ Google Drive โดยตั้งค่า “Anyone with the link can view” โดยไม่ตั้งสิทธิ์
📌 พบว่าอดีตพนักงานยังสามารถเข้าระบบได้
ควรแก้ไข :
✅ เปลี่ยนสิทธิ์แชร์ให้เฉพาะอีเมลภายในองค์กร
✅ ลบบัญชีหรือรีเซ็ตรหัสผ่านผู้ใช้งานที่ลาออก
✅ ใช้ระบบที่รองรับการยืนยันตัวตน (SSO, 2FA)
🧾 Integrity (ความถูกต้องของข้อมูล)

คืออะไร : ข้อมูลต้องถูกต้อง ไม่ถูกเปลี่ยนแปลงโดยไม่ได้รับอนุญาต

ถ้าเจอปัญหาแบบนี้ :
📌 ข้อมูลในไฟล์ Excel ถูกแก้ผิด หรือหายไปโดยไม่ทราบสาเหตุ
📌 รายงานถูกแก้ชื่อ / ตัวเลขโดยไม่ได้แจ้ง
ควรแก้ไข :
✅ เปิดใช้ระบบบันทึกประวัติ (Version History, Audit Log)
✅ กำหนดสิทธิ์แก้ไขเป็นเฉพาะเจ้าของ
✅ สำรองไฟล์ก่อนส่งหรือแชร์ให้ผู้อื่น
🖥️ Availability (ความพร้อมใช้งาน)

คืออะไร : ระบบ/ข้อมูลสามารถใช้งานได้ทันทีเมื่อจำเป็น

ถ้าเจอปัญหาแบบนี้ :
📌 Server ล่มระหว่างประชุมออนไลน์
📌 ไม่สามารถเข้าระบบขออนุมัติได้ในเวลางาน
ควรแก้ไข :
✅ แจ้ง IT เพื่อเช็กความเสถียรของระบบหรือโครงสร้างพื้นฐาน
✅ ควรมีแผนสำรอง (แอปสำรอง, Offline Template)
✅ หมั่นทดสอบระบบล่วงหน้า
🦠 Malware (มัลแวร์)

คืออะไร : ไฟล์หรือซอฟต์แวร์อันตราย เช่น ไวรัส, โทรจัน, แรนซัมแวร์

ถ้าเจอปัญหาแบบนี้ :
📌 คอมพิวเตอร์ทำงานช้า เปิดไฟล์ไม่ได้
📌 มีหน้าจอแปลกๆ ขึ้นมา เช่นเรียกค่าไถ่
ควรแก้ไข :
✅ ตัดการเชื่อมต่ออินเทอร์เน็ตทันที
✅ แจ้ง IT เพื่อสแกนระบบ
✅ หลีกเลี่ยงการเปิดไฟล์แนบจากแหล่งที่ไม่รู้จัก
📧 Phishing (ฟิชชิง)

คืออะไร : อีเมลหรือเว็บไซต์ปลอมหลอกให้กรอกข้อมูล

ถ้าเจอปัญหาแบบนี้ :
📌 ได้อีเมลที่ดูเหมือนจาก HR/ธนาคาร แต่โดเมนผิด เช่น hr-support@fake.com
📌 ลิงก์ในอีเมลพาไปหน้าล็อกอินที่ไม่ใช่ของจริง
ควรแก้ไข :
✅ ห้ามคลิกลิงก์หรือกรอกข้อมูล
✅ แจ้งผู้ดูแลระบบเพื่อตรวจสอบและแจ้งเตือนพนักงานคนอื่น
✅ ใช้ปุ่ม “Report phishing” บนอีเมล (Gmail/Outlook)
🔑 รหัสผ่านอ่อนแอ (Weak Password)

คืออะไร : คืออะไร: รหัสผ่านที่เดาง่าย เช่น 123456 หรือ วันเกิด

ถ้าเจอปัญหาแบบนี้ :
📌 พบว่าบัญชีตนเองหรือคนอื่นถูกเข้าใช้โดยไม่รู้ตัว
📌 รหัสผ่านหลุดเพราะตั้งเหมือนกันหลายระบบ
ควรแก้ไข :
✅ เปลี่ยนรหัสผ่านทันที
✅ ตั้งรหัสผ่านที่คาดเดายาก (ยาว >12 ตัว)
✅ ใช้ Password Manager (Google, Bitwarden)
🛡️ 2FA (Two-Factor Authentication)

คืออะไร : การยืนยันตัวตนแบบ 2 ขั้น เช่น รหัสผ่าน + OTP

ถ้าเจอปัญหาแบบนี้ :
📌 บัญชีถูกแฮก แม้จะตั้งรหัสผ่านดีแล้ว
📌 ลืมเปิด 2FA ในระบบสำคัญ เช่น Email, HR
ควรแก้ไข :
✅ เข้าไปเปิดใช้งาน 2FA ทันทีในระบบที่รองรับ
✅ ใช้แอป Authenticator แทน OTP ผ่าน SMS (ปลอดภัยกว่า)
✅ บันทึกรหัส backup สำหรับการเข้าใช้งานหากมือถือหาย
